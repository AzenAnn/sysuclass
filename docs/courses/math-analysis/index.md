## 学什么

不同院系和学期的顺序可能不同，但数学分析通常围绕以下主线展开：

1. **极限与连续**：把直觉性的“趋近”写成可验证的论证；
2. **一元微分学**：导数、微分、中值定理及其应用；
3. **一元积分学**：定积分、反常积分，以及与级数之间的联系；
4. **级数与函数列**：收敛、逐项运算和一致性问题；
5. **多元微积分**：多元函数的极限、微分与积分。

这只是通用知识地图，具体边界应以当学期大纲为准。

## 推荐学习节奏

### 课前：先找到定义的动机

预习时不必提前做完所有证明。先回答三个问题：本节研究什么对象、要解决什么问题、新定义排除了哪些反例。把不理解的量词和条件圈出来，带着问题听课。

### 课中：记录证明的转折点

不要逐字抄板书。优先记下“为什么想到这一步”、使用了哪个条件、失败的直觉是什么。定义与定理的精确表述可以课后对照教材补齐。

### 课后：用四层结构整理

1. 定义能否不看书完整写出；
2. 定理的条件与结论能否互相区分；
3. 典型例题能否独立复现；
4. 能否构造一个反例说明某个条件不可删除。

数学分析的困难常常不在计算，而在量词、条件和论证链条。错题应标注错误类型，例如“忽略定义域”“把逐点收敛当一致收敛”“引用定理时条件未满足”，比只抄正确答案更有复用价值。

- 按“定义 → 关键定理 → 典型证明 → 反例 → 计算方法”整理每章一页索引；
