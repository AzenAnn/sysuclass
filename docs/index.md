# 中山大学课程攻略共享计划

> 把一届同学走过的路，变成下一届同学找得到、看得懂、还能继续补充的路标。

本项目收集同学原创或已明确获授权的课程攻略、学习笔记、复习提纲、历年资料索引与勘误。资料按课程长期归档，可在页面顶部直接搜索。

!!! warning "非官方说明"
    本项目由同学自发维护，与中山大学官方无隶属关系。课程安排、考核方式和教材可能随院系、教师与学期变化，请以当学期教学大纲和任课教师通知为准。

## 我们收录什么

- 同学原创的课程攻略、学习方法、笔记和复习提纲；
- 经作者明确许可再分发的资料；
- 已由权利人以开放许可发布的内容；
- 试卷题型、考核经验和勘误等可核验信息（不得涉及正在进行的考核）。

每一份文件都应在课程的“资料清单”中登记来源、适用学期和许可状态。只有文件、没有说明的“资料堆”不会被合并。

## 不收录什么

- 盗版或付费电子书、破解软件、付费题库；
- 未经许可复制的教师课件、课程平台文件和答案；
- 带有姓名、学号、手机号、邮箱、成绩等个人信息的文件；
- 泄露中的考试、作业代做或违反学术诚信的内容；
- 对教师或同学的人身攻击、未经证实的传闻。

## 项目状态

项目处于初始化阶段，目录与规则会随着真实投稿继续调整。目前已有[数学分析](courses/math-analysis/index.md)和[高等代数](courses/advanced-algebra/index.md)两门示范课程，欢迎从一条勘误、一份原创笔记或一段课程经验开始贡献。
