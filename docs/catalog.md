# 课程目录

课程按“院系/培养方案中的常用名称”归档；同名但内容差异显著的课程，可在课程页内按院系或课程代码分栏，而不是直接复制多个模糊目录。

所有课程页都归在本站顶部的“课程目录”栏目下，并显示在该栏目的侧边导航中；新增课程不会再占用顶部标签。
