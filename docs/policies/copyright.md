# 版权、隐私与学术诚信

本页是投稿审核底线，不是法律意见。无法确认时，宁可只写资料说明与合法来源链接，也不要把文件复制进仓库。

## 可以直接上传

- 你本人原创，且愿意按本项目许可分享的文字、笔记、图表和题解；
- 权利人书面同意在本项目公开再分发的材料（保留授权记录）；
- 明确标注了开放许可，且本项目使用方式符合该许可的资料；
- 已进入公有领域、来源与状态可核验的材料。

## 需要先获得许可

- 教师制作的课件、讲义、题库和课程平台下载文件；
- 同学的笔记、整理、录音、录像和作业；
- 学院、社团或其他项目制作的资料；
- 试卷与答案（“考完了”不等于可以公开再分发）。

## 不接受

- 盗版电子书、影印教材、破解软件和绕过付费墙得到的内容；
- 未获许可的录音录像、教师课件和课程平台内容；
- 考试泄题、仍在使用的内部题库、代写代做材料；
- 含第三人个人信息而未完成脱敏的资料；
- 恶意评价、骚扰、歧视、诽谤或无法核验的指控。

## 脱敏要求

上传前逐页检查封面、页眉页脚、批注、文件属性和图片角落，移除：

- 姓名、学号、证件号、手机号、邮箱、宿舍地址；
- 成绩、排名、签名、二维码、群号与内部系统链接；
- 能够间接识别个人的班级名单和批改记录。

PDF/Office 文件的“作者”“最后修改者”等元数据也可能泄露身份。不会清理时，在 Issue 中说明并请维护者协助，不要直接公开上传。

## 许可标注

贡献者原创内容默认采用 [CC BY-NC-SA 4.0](https://creativecommons.org/licenses/by-nc-sa/4.0/deed.zh-hans)。每份第三方材料必须沿用其自身许可，并在资料清单中写清作者、来源、许可与适用限制。

维护者收到权利人通知后，可先临时下架相关资料再核验。侵权下架请求应通过仓库 Issue 或维护者在项目主页公布的联系渠道提交，并提供能定位文件的信息。
